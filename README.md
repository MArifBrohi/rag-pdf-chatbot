## 🔗 Live Demo
**[👉 Try the app here](https://rag-pdf-chatbot-devarif.streamlit.app/)**

---

## 🚀 Features

- 📁 **Multi-PDF Support** — Upload multiple PDFs simultaneously
- 💬 **Chat History** — Remembers previous questions in conversation
- 📍 **Source Attribution** — Shows exact page numbers used for each answer
- ⚡ **Groq LLaMA 3.3 70B** — Fast and accurate responses
- 🔍 **FAISS Vector Search** — Finds most relevant chunks instantly
- 🌙 **Dark/Light Mode** — Works in both themes
- 🧠 **HuggingFace Embeddings** — Free local embeddings, no API cost

---

## 🛠️ Tech Stack

| Component | Technology |
|-----------|-----------|
| Frontend | Streamlit |
| LLM | Groq LLaMA 3.3 70B |
| Embeddings | HuggingFace (all-MiniLM-L6-v2) |
| Vector Store | FAISS |
| PDF Processing | PyPDF |
| Framework | LangChain |
| Language | Python 3.10+ |
